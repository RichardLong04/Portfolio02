ISUD

Here is my first group project at Illinois Tech. We gave really bad site a makeover, using HTML5/CSS3/JavaScript. Technical Marketing Inc.

TMI was my first freelance project. My client wanted their old outdated website to be given a modern make over. I had to incorporate the existing content into the a new layout that was responsive for mobile devices. However, after I decided to use a blank bootstrap template and create my own layout, everything scaled down nicely and the customer was pleased.
